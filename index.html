Hoy aprendi

    1. La etiqueta "h1" la cual es para establecer títulos.
    2. Si quiero escribir texto normal, puedo utilizar "p".
    3. También pude establecer una forma de navegar en el blog con "a href=el_lugar".
    4. Para agregar este formato de lista, implementé "ol" que me da un orden a la lista, seguido de "li".
    5. Posteriormente descubrí "article", que es un elemento independiente, aún no sé cómo aplicarlo bien.
    6. Usé "section" para agrupar contenido relacionado, como la sección de inicio y la sección del blog.
    7. Para mostrar una imagen, usé "img" con "src" para la URL de la imagen y "alt" para el texto alternativo.
    8. Incrusté un video de YouTube usando "iframe", que permite mostrar contenido externo en mi página.
    9. Creé una tabla con "table", usando "tr" para las filas, "th" para los encabezados y "td" para las celdas.
    10. Usé "header" para definir la cabecera de mi página, que incluye el menú de navegación.
    11. El menú de navegación lo creé con "nav", que contiene enlaces ("a") a diferentes secciones.
    12. Para el pie de página, usé "footer", donde incluí mi nombre y un enlace a GitHub..
    13. Usé "main" para envolver el contenido principal de la página, lo que ayuda a los motores de búsqueda a entender la estructura.
    14. Para hacer la página responsive, incluí "meta name=viewport" en el "head".
